# China Briefing

Practical China travel planning pages for independent travelers.

This prototype helps visitors understand how to plan a self-guided China trip: which city to start with, what apps to install, how to think about booking paths, how to avoid holiday crowd problems, and how to explore Shanghai without building a heavy ticketing platform.

## Current Pages

- `index.html` - homepage with crowd forecast, city guide entry points, app toolkit, booking guidance, FAQ, and travel alert signup.
- `cities/shanghai.html` - Shanghai city guide with categorized attractions, food recommendations, transport notes, costs, booking links, airport context, and stay advice.
- `guides/book-hotels-in-china.html` - hotel booking guide for independent travelers.
